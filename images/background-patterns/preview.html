How to Use

You can use these background patterns in your CSS by referencing the SVG files: 

/* To use the hero background */
.your-element {
  background-image: url('./hero-background.svg');
  background-size: cover;
}

/* To use the community voices background */
.your-element {
  background-image: url('./community-voices-background.svg');
  background-size: cover;
}

/* To use the community voices background without gradient */
.your-element {
  background-image: url('./community-voices-lines-only.svg');
  background-size: cover;
}

/* To use the our purpose background */
.your-element {
  background-image: url('./our-purpose-background.svg');
  background-size: cover;
}

/* To use just the patterns */
.your-element {
  background-image: url('./pattern-dots.svg');
  background-repeat: repeat;
}

.your-element {
  background-image: url('./pattern-lines.svg');
  background-repeat: repeat;
}
