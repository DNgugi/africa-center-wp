Typography Demo - Africa Center Theme

This is a demonstration of the typography system implemented for the Africa Center WordPress theme. The theme now includes comprehensive typography styles that provide excellent readability and visual hierarchy.

Main Heading (H2)

This is a paragraph following an H2 heading. The theme uses Open Sans for body text, which provides excellent readability at various sizes. Links are styled with the theme's primary light blue color and include hover effects.

Section Heading (H3)

H3 headings use the terracotta color from the theme's color palette. This creates a nice visual hierarchy while maintaining readability. The typography system is fully responsive and will adjust appropriately on different screen sizes.

Subsection Heading (H4)

H4 headings use the green color from the theme's palette. Each heading level has appropriate spacing and font weights to create a clear visual hierarchy.

Minor Heading (H5)

H5 headings return to the primary blue color but in a smaller size.

Small Heading (H6)

H6 headings are uppercase with letter spacing and use the primary blue color.

Color Palette

The typography system uses the theme's comprehensive color palette:

  • Headings: Primary Blue (#1E5091) and Terracotta (#C75D32)
  • Body Text: Neutral Darkest (#2C2927)
  • Links: Primary Light Blue (#3A7CC9)
  • Accents: Primary Green (#4D7F41) and Ochre (#D9A23B)

Responsive Design

All typography elements are responsive and will scale appropriately on different screen sizes. Font sizes increase on larger screens while maintaining readability on mobile devices.
